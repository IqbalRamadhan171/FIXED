Tutorial Cara Top-up

  1. Masukkan User ID dan Zone ID Anda, Contoh : 667632423 ( 5246 )
  2. Pilih Nominal Diamonds yang kamu inginkan
  3. Selesaikan pembayaran
  4. Diamonds akan ditambahkan ke akun Mobile Legends kamu

Informasi Penting!

  1. Pastikan Level Akun Mobile Legends kamu sudah mencapai level 5
  2. Kamu tidak bisa Top Up Weekly Diamond jika sudah 0/10 (70 Hari) kalian bisa check di bagian weekly diamond masing masing akun
  3. Contoh : Jika masih 3/10 berarti kalian masih bisa top up 3x lagi
  4. HARAP DI CEK DLU APAKAH WDP KALIAN LIMIT ATAU TIDAK. KRN RIBET KALO UDAH LIMIT MASIH MAKSA BELI

Lengkapi Data

Untuk mengetahui User ID Anda, silahkan klik menu profile dibagian kiri atas pada menu utama game. User ID akan terlihat dibagian bawah Nama karakter game Anda. Silahkan masukan User ID dan Server ID Anda untuk menyelesaikan transaksi. Contoh: 12345678(1234), maka ID = 12345678 dan Server = 1234.
